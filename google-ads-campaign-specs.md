## Conversion Tracking (Already Configured in GTM)

| Conversion Action | Event Name | Type | Use in Ads |
|---|---|---|---|
| Form submission (thank-you page) | `generate_lead` | Primary | Smart Bidding optimizes toward this |
| Phone call click | `phone_call_click` | Secondary | Reported but not used for bidding |

**To activate in Google Ads:**
1. Go to Goals > Conversions > Summary
2. Click + New conversion action > Import > Google Analytics 4
3. Select `generate_lead` > mark as Primary
4. Select `phone_call_click` > mark as Secondary (observation only)

## Optimization Schedule

### Week 1-2 (Learning Phase)
- Monitor daily. Do NOT change bids or budgets.
- Add negative keywords from Search Terms report daily.
- Ensure ads are approved and serving.

### Week 3-4
- Pause any ad group with CTR < 1.5%
- Pause keywords with 100+ impressions and 0 clicks
- Add converting search terms as exact match keywords
- Review device performance — adjust mobile bid if needed

### Monthly (Ongoing)
- Review Search Terms report — add negatives, promote winners
- A/B test new ad copy (change one element at a time)
- Review geographic performance — shift budget to best-performing areas
- Check competitor ads using Auction Insights
- Adjust budgets based on seasonal demand (increase Oct-Mar, reduce Apr-Sep)

### Quarterly
- Review landing page conversion rates — test improvements
- Evaluate campaign ROI against actual closed revenue
- Consider adding Performance Max campaign if Search is maxing out
- Expand to new service areas if profitable
